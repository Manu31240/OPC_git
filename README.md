## les listes

* élément un
* élément deux
  * sous-point élément deux 

la commande:</br>
	\* élément un</br>
	\* élément deux</br>
  	\  * sous-point élément deux</br> 

Faire une liste ordonnée en utilisant des nombres à la place :
1. Element un 
2. Element deux 

la commande :</br>
	\1. Element un</br>
	\2. Element deux</br>

Faire une liste de tâches en incluant des crochets :
- [ ] Element un 
- [ ] Element deux 
- [x] Element terminé 

Rendu sous la forme :
	•  Element un </br>
	•  Element deux
	•  Element terminé
